# AIOps-Labs
Methods and Tools for AIOps: Observability

## Course Overview

This course is focused on **Observability** for cloud-based applications, covering instrumentation and capture of data needed to apply AI methods for anomaly detection and correction. The curriculum presents fundamental DevOps concepts including Docker, CI/CD pipelines, and microservices architectures used in hybrid cloud deployments.

### Core Topics
- **Instrumentation & Monitoring**: Methods for instrumenting applications to observe their behavior
- **Data Storage & Visualization**: Storing and displaying observability data
- **DevOps Fundamentals**: Docker, CI/CD pipelines, and microservices architectures
- **Cloud Platforms**: Hybrid cloud deployments and containerization

### Tools & Technologies
- **Docker**: Containerization and orchestration
- **Prometheus**: Time-series database for metrics
- **Grafana**: Dashboard and visualization platform
- **Dash**: UI framework for interactive dashboards
- **Locust**: Load and fault injection testing
- **Flask**: HTTP framework for building APIs

### Prerequisites
While there are no formal prerequisites, a strong background in Python programming and exposure to DevOps and cloud platforms (Docker, microservices) is helpful.

## Lab 1: Getting Started with Docker & Containerization

Lab 1 introduces foundational concepts and hands-on experience with containerization using Docker. This lab serves as the building block for the AIOps class in the Fall semester.

### Lab 1 Activities

**Objectives:**
- Understand Docker fundamentals and containerization concepts
- Build and run containerized applications
- Work with multi-container applications using Docker Compose
- Implement data persistence in containerized environments
- Explore microservices architecture patterns

**Key Components:**
1. **Docker Basics**: Building Docker images, running containers, and managing container lifecycle
2. **Multi-Container Applications**: Using Docker Compose to orchestrate multiple services
3. **Data Persistence**: Implementing SQLite and MySQL for data storage in containers
4. **Application Development**: Building a sample todo application with frontend and backend services
5. **Microservices Exploration**: Introduction to the Boutique microservices demo application

**Deliverables:**
- Containerized todo application with persistent storage
- Docker Compose configuration for multi-container setup
- Understanding of container networking and volume management
- Foundation for advanced observability topics in subsequent labs
